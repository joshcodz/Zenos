> **Zenos** is a highly polished, interactive productivity dashboard designed to eliminate distractions. Featuring a breathtaking "Fabulous Glassy Aura" aesthetic, dynamic widgets, and a cinematic "Zen Mode," Zenos turns your browser into a high-end operating system engineered for deep focus.

---

## 🌟 Key Features

### 🧠 The Command Capsule
Say goodbye to clunky navigation. The top-center **Command Capsule** functions as a dynamic island. Hover or click to expand it and reveal your essential productivity tools:
- **Focus Timer**: A built-in Pomodoro tracker with session analytics.
- **Tasks**: A glassy, draggable to-do list to keep track of your goals.
- **Weather & Motivation**: Live local weather and daily rotating stoic quotes.

### 🧘‍♂️ Cinematic Zen Mode
Enter ultimate deep work mode with a single click. Toggling **Zen Mode** gracefully fades out the entire UI, dims the background, and leaves only a massive, perfectly centered clock and your active Pomodoro timer. Zero distractions.

### ⌨️ Intelligent Command Palette
Press `Ctrl+K` (or `Cmd+K`) from anywhere to summon a Spotlight-style global search bar. Instantly search for and toggle widgets, activate Zen Mode, or mute audio without ever touching your mouse.

### 💊 The Media Pill
A sleek, unified control center tucked away in the bottom corner. Expand the Media Pill to:
- Change your environment with **Spaces** (curated aesthetic video backgrounds).
- Mix custom **Ambient Audio** (Rain, Cafe, Ocean Waves, etc.).
- Control your **Spotify integration** (coming soon).

### ✨ True Glassmorphism Aesthetic
Zenos uses cutting-edge CSS design. Featuring intense backdrop blurs, dynamic animated aurora gradients, specular edge highlights, and tactile `framer-motion` physics that make widgets react when you drag them.

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/joshcodz/Zenos.git
   cd Zenos
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Experience Zenos**
   Open `http://localhost:5173` in your browser. 
   *(Note: To experience the cinematic onboarding flow again, clear your browser's local storage for localhost).*
